# PharmacyPro-Manager
A Desktop-based Pharmacy Management System with Stock Entry, Sales Tracking, and PDF Ledger Generation.
Features
Stock Management:** Track medicine purchases from suppliers.
Sales Tracking:** Dedicated sales ledger for hospitals.
PDF Invoicing:** Generate professional PDF reports for both Purchase and Sales.
Modern UI:** Built with CustomTkinter for an attractive user experience.

##  Built With
Language: Python 3.14
GUI: CustomTkinter
Database: SQLite3
Reports: ReportLab (PDF)

 How to Install
1. Clone the repo: `git clone https://github.com/YourUsername/PharmacyPro-Manager.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run: `python main.py`
